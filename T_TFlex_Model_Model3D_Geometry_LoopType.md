# LoopType - перечисление  
  
---  
  
Тип цикла

**Пространство имён:** [TFlex.Model.Model3D.Geometry](N_TFlex_Model_Model3D_Geometry.md)**Сборка:** TFlexAPI3D (в TFlexAPI3D.dll) Версия: 17.1.20.0

| Имя члена | Значение | Описание |
| --- | --- | --- |
| Vertex | 0 | цикл без ребёр |
| Wire | 1 | цикл, без внутренности, например, от проволочного тела |
| Outer | 2 | обычный внешний цикл |
| Inner | 3 | обычный внутренний цикл |
| Winding | 4 | витой цикл периодической поверхности, например, окружность на циллиндре или торе |
| InnerSing | 5 | цикл, который представляет собой дырку вокруг поверхностной сингулярности, например, отсечение вершины конуса |
| LikelyOuter | 6 | подобен внешнему циклу на дважды замкнутой поверхности |
| LikelyInner | 7 | подобен дырке на дважды замкнутой поверхности |
| Unclear | 8 | цикл, который делит периодическую вырожденную поверхность на две (содержит только один полюс) |
| Error | 9 | неверный цикл или сбой алгоритма |
