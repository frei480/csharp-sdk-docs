# ObjectSelectControlMode - перечисление  
    
Режим отображения ObjectSelectControl

**Пространство имён:** [TFlex.Dialogs](N_TFlex_Dialogs.md)**Сборка:** TFlexCommandAPI (в TFlexCommandAPI.dll) Версия: 17.1.20.0

```csharp
public enum ObjectSelectControlMode
```




| Имя члена | Значение | Описание |
| --- | --- | --- |
| SingleObject | 0 | Может быть выбран максимум один объект. Контрол всегда свёрнут. |
| MultiObjectCollapsed | 1 | Может быть выбрано любое количество объектов. Контрол свёрнут. |
| MultiObjectExpanded | 2 | Может быть выбрано любое количество объектов. Контрол развёрнут. |
