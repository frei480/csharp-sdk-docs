# BaseControlDispose(Boolean) - метод  
    
Освобождает неуправляемые ресурсы, используемые объектом [BaseControl](T_TFlex_Dialogs_BaseControl.md), а при необходимости освобождает также управляемые ресурсы

**Пространство имён:** [TFlex.Dialogs](N_TFlex_Dialogs.md)**Сборка:** TFlexCommandAPI (в TFlexCommandAPI.dll) Версия: 17.1.20.0

```csharp
protected override void Dispose(
	bool A_0
)
```




#### Параметры

A_0 [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    Значение true позволяет освободить управляемые и неуправляемые ресурсы; значение false позволяет освободить только неуправляемые ресурсы.
