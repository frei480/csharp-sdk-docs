# DeleteOptions - свойства  
    
Тип [DeleteOptions](T_TFlex_Model_DeleteOptions.md) предоставляет следующие члены.

| --- | --- | --- |
|  | [DeleteObjectsOnLayer](P_TFlex_Model_DeleteOptions_DeleteObjectsOnLayer.md) | Используется методом DeleteObjects |

|  | [DisableMacrosSmartFragments](P_TFlex_Model_DeleteOptions_DisableMacrosSmartFragments.md) | Отключить запуск макроса у смарт-фрагментов (событие удаления фрагмента) |
|  | [Exclude](P_TFlex_Model_DeleteOptions_Exclude.md) | Исключить удаляемые элементы из модели |
|  | [Silent](P_TFlex_Model_DeleteOptions_Silent.md) | Режим "без вопроса" |
