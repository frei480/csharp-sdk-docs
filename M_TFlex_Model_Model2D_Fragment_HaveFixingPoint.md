# FragmentHaveFixingPoint - метод  
    
Проверка существования точки привязки фрагмента, привязанного при помощи переменных привязки

**Пространство имён:** [TFlex.Model.Model2D](N_TFlex_Model_Model2D.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public bool HaveFixingPoint(
	int index
)
```




#### Параметры

index [Int32](https://learn.microsoft.com/dotnet/api/system.int32)
    Номер точки привязки. Может иметь значение от 1 до 9

#### Возвращаемое значение

[Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)true, если точка привязки фрагмента с указанным номером существует, иначе false
