# ApplicationDoEvents - метод  
    
Обрабатывает все сообщения Windows, которые в данный момент находятся в очереди сообщений.

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public static void DoEvents()
```




Метод позволяет приложению обрабатывать другие события, которые могут возникнуть при выполнии кода
