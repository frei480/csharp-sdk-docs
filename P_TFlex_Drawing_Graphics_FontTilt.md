# GraphicsFontTilt - свойство  
    
Установка угла наклона шрифта

**Пространство имён:** [TFlex.Drawing](N_TFlex_Drawing.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

#### Значение свойства

[Double](https://learn.microsoft.com/dotnet/api/system.double)

Угол наклона учитывается только при выводе шрифта формата SHX. Угол измеряется в градусах. По умолчанию угол наклона шрифта равен 90.
