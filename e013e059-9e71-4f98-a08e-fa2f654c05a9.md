# Получение объектов  
    
Для получения объектов модели предназначены следующие методы:

  * [GetObjectByName](M_TFlex_Model_Document_GetObjectByName.md) позволяет получить объект документа по его имени. У объекта может существовать имя, являющееся строкой. Имя, присвоенное объекту, уникально в пределах данного документа. 
  * [GetObjectByID](M_TFlex_Model_Document_GetObjectByID.md) позволяет получить объект по его идентификатору. Идентификатор объекта является беззнаковым целым числом, которое является уникальным в пределах данного документа. 



Примечание  Также существует целый , позволяющих перебирать объекты документа, находящиеся в различных контейнерах   
  
Ниже приведен пример кода функции, которая получает объекты по идентификатору и по имени.
    
    
    private void ObjectsSearch(Document document)
    {
      //Получение объекта по идентификатору
      ModelObject object1 = document.GetObjectByID(0x3000017);
      //Получение объекта по имени
      ModelObject object2 = document.GetObjectByName("Выталкивание_0");
    }
    
    
    Private Sub ObjectsSearch(ByVal document As Document)
      'Получение объекта по идентификатору
      object1 As ModelObject
      object1 = document.GetObjectByID(0x3000017)
      'Получение объекта по имени
      object2 As ModelObject
      object2 = document.GetObjectByName("Выталкивание_0")
    End Sub
