# PopupToolBarButtonSetAdd(Int32) - метод  
    
Добавляет в этот набор кнопку с указанным идентификатором

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public bool Add(
	int id
)
```




#### Параметры

id [Int32](https://learn.microsoft.com/dotnet/api/system.int32)
    Идентификатор кнопки

#### Возвращаемое значение

[Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)true, если кнопка была добавлена в этот набор; false, если указанный идентификатор недействителен или используется одной из кнопок, уже имеющихся в этом наборе
