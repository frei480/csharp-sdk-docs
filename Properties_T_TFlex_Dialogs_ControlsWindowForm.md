|  | [MinHeight](P_TFlex_Dialogs_ControlsWindowForm_MinHeight.md) | Минимальная высота формы |

|  | [SupportsFloatingWindow](P_TFlex_Dialogs_ControlsWindowForm_SupportsFloatingWindow.md) | Форма может быть показана в отдельном плавающем окне |
