# GeometryData - свойства  
    
Тип [Path3DGeometryData](T_TFlex_Model_Model3D_Path3D_GeometryData.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Axis](P_TFlex_Model_Model3D_Path3D_GeometryData_Axis.md) | Если путь лежит на прямой, то можно получить её направление |
|  | [Box](P_TFlex_Model_Model3D_Path3D_GeometryData_Box.md) | Получить границы пути |
|  | [Contour](P_TFlex_Model_Model3D_Path3D_GeometryData_Contour.md) | Получить контур |
|  | [Curve](P_TFlex_Model_Model3D_Path3D_GeometryData_Curve.md) | Получить кривую, на которой лежит кривая |
|  | [LineDirection](P_TFlex_Model_Model3D_Path3D_GeometryData_LineDirection.md) | Если ребро лежит на прямой, то можно получить её направление |
|  | [Plane](P_TFlex_Model_Model3D_Path3D_GeometryData_Plane.md) | Если путь лежит в плоскости, то можно получить эту плоскость |
|  | [PlaneDirection](P_TFlex_Model_Model3D_Path3D_GeometryData_PlaneDirection.md) | Если путь лежит в плоскости, то можно получить нормаль к этой плоскости |
|  | [Point](P_TFlex_Model_Model3D_Path3D_GeometryData_Point.md) | Если путь лежит на окружности или эллипсе, то можно получить их центр |
|  | [Wire](P_TFlex_Model_Model3D_Path3D_GeometryData_Wire.md) | Получить проволочную модель |
