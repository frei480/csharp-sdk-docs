# ImportFrom3dOption - перечисление  
  
---  
  
Возможные варианты использования некоторых опций импорта

**Пространство имён:** [TFlex.Model](N_TFlex_Model.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public enum ImportFrom3dOption
```
```vb
Public Enumeration ImportFrom3dOption
```
```cpp
public enum class ImportFrom3dOption
```


| Имя члена | Значение | Описание |
| --- | --- | --- |
| Auto | 0 | Решение о том будет ли использоваться опция будет принято автоматически исходя из формата импортируемого файла |
| Yes | 1 | Опция будет использоваться всегда для любых форматов |
| No | 2 | Опция не будет использоваться никогда ни для каких форматов |
| YesWithOrientation | 3 | Опция будет использоваться всегда для любых форматов. Дополнительно применять лечение ориентаций. Используется для Step |
