# DrawingView - свойства  
  
---  
  
Тип [DrawingView](T_TFlex_Model_Model2D_DrawingView.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Angle](P_TFlex_Model_Model2D_DrawingView_Angle.md) | Угол поворота изображения. При установке значения привязка по вектору отменяется |
|  | [Attributes](P_TFlex_Model_ModelObject_Attributes.md) | **Устарело.**(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [BoundRect](P_TFlex_Model_Model2D_Object2D_BoundRect.md) | Получение координат прямоугольника, обрамляющего объект(Унаследован от [Object2D](T_TFlex_Model_Model2D_Object2D.md)) |
|  | [BreakSet](P_TFlex_Model_Model2D_DrawingView_BreakSet.md) | Набор разрывов чертёжного вида |
|  | [Clipping](P_TFlex_Model_Model2D_DrawingView_Clipping.md) | Выполнять отсечение изображения по текущему прямоугольнику отсечения (свойство ClippingRectangle) |
|  | [ClippingRectangle](P_TFlex_Model_Model2D_DrawingView_ClippingRectangle.md) | Прямоугольная область отсечения в системе координат отображаемой страницы |
|  | [DisplayName](P_TFlex_Model_ModelObject_DisplayName.md) | Отображаемое название объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Document](P_TFlex_Model_ModelObject_Document.md) | Документ, являющийся родительским для данного объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Editable](P_TFlex_Model_ModelObject_Editable.md) | Объект находится в состоянии редактирования(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [GroupType](P_TFlex_Model_Model2D_DrawingView_GroupType.md) | Тип объекта(Переопределяет [ModelObjectGroupType](P_TFlex_Model_ModelObject_GroupType.md)) |
|  | [HideBreaksLines](P_TFlex_Model_Model2D_DrawingView_HideBreaksLines.md) | Скрывать линии разрывов |
|  | [ID](P_TFlex_Model_ModelObject_ID.md) | **Устарело.** Идентификатор объекта. Идентификатор является уникальным числом для каждого из объектов одного документа(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [IsActive](P_TFlex_Model_Model2D_DrawingView_IsActive.md) | Вид является активным |
|  | [IsDisposed](P_TFlex_Model_ModelObject_IsDisposed.md) | Объект удален из модели(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [IsInModelObjectGroup](P_TFlex_Model_ModelObject_IsInModelObjectGroup.md) | Является ли объект элементом группы(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [IsVisible](P_TFlex_Model_ModelObject_IsVisible.md) | **Устарело.** Это свойство устарело и будет удалено. Пожалуйста, используйте свойство 'Visible'.(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Layer](P_TFlex_Model_Model2D_DrawingView_Layer.md) | Слой, на котором размещается объект |
|  | [LeaderNote](P_TFlex_Model_Model2D_DrawingView_LeaderNote.md) | Объект надписи обозначения (только для выносного вида) |
|  | [Level](P_TFlex_Model_Model2D_DrawingView_Level.md) | Уровень объекта |
|  | [ModelObjectGroup](P_TFlex_Model_ModelObject_ModelObjectGroup.md) | Группа, которая включает данный объект(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Name](P_TFlex_Model_ModelObject_Name.md) | Имя объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Node1](P_TFlex_Model_Model2D_DrawingView_Node1.md) | Первый узел привязки |
|  | [Node2](P_TFlex_Model_Model2D_DrawingView_Node2.md) | Второй узел привязки. Используется только в режиме привязки по вектору |
|  | [ObjectId](P_TFlex_Model_ModelObject_ObjectId.md) | Идентификатор объекта. Идентификатор является уникальным для каждого из объектов одного документа(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [OriginX](P_TFlex_Model_Model2D_DrawingView_OriginX.md) | Координата X базовой точки на отображаемой странице |
|  | [OriginY](P_TFlex_Model_Model2D_DrawingView_OriginY.md) | Координата Y базовой точки на отображаемой странице |
|  | [Page](P_TFlex_Model_Model2D_DrawingView_Page.md) | Страница, на которой размещается элемент(Переопределяет [Object2DPage](P_TFlex_Model_Model2D_Object2D_Page.md)) |
|  | [PageScale](P_TFlex_Model_ModelObject_PageScale.md) | Масштаб страницы объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Parents](P_TFlex_Model_ModelObject_Parents.md) | Контейнер родительских объектов(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Priority](P_TFlex_Model_Model2D_DrawingView_Priority.md) | Приоритет объекта |
|  | [Scale](P_TFlex_Model_Model2D_DrawingView_Scale.md) | Масштаб изображения |
|  | [ShowConstructions](P_TFlex_Model_Model2D_DrawingView_ShowConstructions.md) | Показывать линии построения |
|  | [ShowPage](P_TFlex_Model_Model2D_DrawingView_ShowPage.md) | Отображаемая на виде страница |
|  | [SynchronizePageParameters](P_TFlex_Model_Model2D_DrawingView_SynchronizePageParameters.md) | Синхронизировать параметры страницы |
|  | [Transformation](P_TFlex_Model_Model2D_DrawingView_Transformation.md) | Преобразование, применяемое при выводе отображаемой страницы |
|  | [TransparentActivation](P_TFlex_Model_Model2D_DrawingView_TransparentActivation.md) | Разрешить выполнение автоматической активации |
|  | [Vector](P_TFlex_Model_Model2D_DrawingView_Vector.md) | Вектор привязки |
|  | [ViewFixingMode](P_TFlex_Model_Model2D_DrawingView_ViewFixingMode.md) | Режим привязки |
|  | [ViewSymbol](P_TFlex_Model_Model2D_DrawingView_ViewSymbol.md) | Объект обозначения вида |
|  | [Visible](P_TFlex_Model_Model2D_DrawingView_Visible.md) | Изображение вида показывать |
|  | [X1](P_TFlex_Model_Model2D_DrawingView_X1.md) | Координата X первой точки привязки |
|  | [X2](P_TFlex_Model_Model2D_DrawingView_X2.md) | Координата X второй точки привязки. Используется только в режиме привязки по вектору |
|  | [Y1](P_TFlex_Model_Model2D_DrawingView_Y1.md) | Координата Y первой точки привязки |
|  | [Y2](P_TFlex_Model_Model2D_DrawingView_Y2.md) | Координата Y второй точки привязки. Используется только в режиме привязки по вектору |
