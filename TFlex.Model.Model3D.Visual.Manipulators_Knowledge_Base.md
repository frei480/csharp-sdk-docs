# База знаний по классам в пространстве имён TFlex.Model.Model3D.Visual.Manipulators

## LengthDiameterManipulator
**Описание:** Манипулятор для изменения длины и диаметра.
**Свойства:** Length, Diameter, Position
**События:** ValueChanged

## RotationManipulator
**Описание:** Манипулятор для вращения объектов.
**Свойства:** Axis, Angle
**События:** RotationChanged

## ShaftManipulator
**Описание:** Комплексный манипулятор для валов.
**Методы:** Initialize(), Activate(), Deactivate()</content>
