# FullRegenerationWindowItem - свойства  
    
Тип [FullRegenerationWindowItem](T_TFlex_FullRegenerationWindowItem.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Id](P_TFlex_FullRegenerationWindowItem_Id.md) | Идентификатор, уникальный для каждого элемента в рамках плагина |
|  | [IsChecked](P_TFlex_FullRegenerationWindowItem_IsChecked.md) | Начальное состояние выбора. По умолчанию истинно. |
|  | [Name](P_TFlex_FullRegenerationWindowItem_Name.md) | Отображаемое название элемента в диалоге |
|  | [ParentId](P_TFlex_FullRegenerationWindowItem_ParentId.md) | Идентификатор родителя. Если задан, то текущий элемент будет активен только в том случае, если в диалоге у родительского элемента задан флаг. |
