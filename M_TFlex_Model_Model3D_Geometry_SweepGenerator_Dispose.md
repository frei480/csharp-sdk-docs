# SweepGeneratorDispose(Boolean) - метод  
    
Освобождает неуправляемые ресурсы, используемые объектом [SweepGenerator](T_TFlex_Model_Model3D_Geometry_SweepGenerator.md), а при необходимости освобождает также управляемые ресурсы

#### Параметры

A_0 [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    Значение true позволяет освободить управляемые и неуправляемые ресурсы; значение false позволяет освободить только неуправляемые ресурсы.
