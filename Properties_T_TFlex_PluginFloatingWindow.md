# PluginFloatingWindow - свойства  
    
Тип [PluginFloatingWindow](T_TFlex_PluginFloatingWindow.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [ActiveDocument](P_TFlex_PluginFloatingWindow_ActiveDocument.md) | Возвращает активный документ |
|  | [Caption](P_TFlex_PluginFloatingWindow_Caption.md) | Заголовок окна |
|  | [Control](P_TFlex_PluginFloatingWindow_Control.md) | Возвращает клиентскую часть всплывающего окна |
|  | [Height](P_TFlex_PluginFloatingWindow_Height.md) | возвращает или устанавливает высоту окна |
|  | [Icon](P_TFlex_PluginFloatingWindow_Icon.md) | Иконка окна |
|  | [ID](P_TFlex_PluginFloatingWindow_ID.md) | Возвращает идентификатор окна |
|  | [Owner](P_TFlex_PluginFloatingWindow_Owner.md) | Возвращает объект приложения, в котором было создано плавающее окно |
|  | [PropertiesForm](P_TFlex_PluginFloatingWindow_PropertiesForm.md) | Возвращает клиентскую часть всплывающего окна |
|  | [TopLeftPoint](P_TFlex_PluginFloatingWindow_TopLeftPoint.md) | Координата левого верхнего угла |
|  | [Visible](P_TFlex_PluginFloatingWindow_Visible.md) | Свойство видимости окна |
|  | [Width](P_TFlex_PluginFloatingWindow_Width.md) | возвращает или устанавливает ширину окна |
