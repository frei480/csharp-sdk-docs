# Управление конфигурациями библиотек  
    
Для управления конфигурациями библиотек в Open API разработан специальный класс [LibraryConfigurations](T_TFlex_LibraryConfigurations.md) и статический метод [LibraryConfigurations](P_TFlex_Application_LibraryConfigurations.md) для получения объекта этого класса. Данный класс представляет собой набор конфигураций библиотек, позволяющий открывать, закрывать, создавать новые конфигурации библиотек, а также получать объекты самих конфигураций библиотек (класс [LibraryConfiguration](T_TFlex_LibraryConfiguration.md)) для управления ими.

Ниже приведен пример кода, сохраняющего все открытые в системе конфигурации библиотек.    
```csharp
private void CloseAllLibaries()
{
	//Перебор всех конфигураций библиотек в приложении
	foreach(LibraryConfiguration configuration in Application.LibraryConfigurations)
	{
		//Сохранение каждой конфигурации
		configuration.Save();
	}
}
```
