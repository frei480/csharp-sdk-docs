# ControlMouseDblClick - делегат  
    
Делегат представляющий метод, который будет обрабатывать событие, происходящее, когда визуальный элемент управления дважды щелкается мышью.

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public delegate void ControlMouseDblClick(
	Object sender,
	ControlEventArgs e
)
```

    Источник события
