|  | [Path](P_TFlex_Model_Model3D_RelationPoint_Element_Path.md) | Получить путь, относительно которого или на котором ищется ближайшая точка |
|  | [Point](P_TFlex_Model_Model3D_RelationPoint_Element_Point.md) | Второй элемент как точка, относительно которой ищется ближайшая точка на первом элементе |
