# GraphicsFontAngle - свойство  
    
Установка угла поворота текста

**Пространство имён:** [TFlex.Drawing](N_TFlex_Drawing.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public double FontAngle { set; }
```




#### Значение свойства

[Double](https://learn.microsoft.com/dotnet/api/system.double)

Угол поворота измеряется от горизонтали. Положительное значение угла соответствует повороту против часовой стрелки. По умолчанию значение этого параметра равно 0. Альтернативным способом задания угла поворота текста является метод [SetFontRotation(Double, Double)](M_TFlex_Drawing_Graphics_SetFontRotation.md)
