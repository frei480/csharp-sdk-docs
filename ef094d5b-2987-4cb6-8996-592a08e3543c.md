# Управление вспомогательными окнами  
    
В системе имеется ряд вспомогательных окон (окно диагностики, окно общего вида, структура 3D модели и т.д.). Для управления такими окнами реализован специальный класс ServiceWindow. Доступ к объектам данного класса осуществляется через . 

Ниже приведен пример кода, показывающего все вспомогательные окна.
    
    
    private void ShowAllServiceWindows()
    {
        //Перебор всех вспомогательных окон в приложении
        foreach(ServiceWindow window in Application.Window.ServiceWindows)
        {
            //Показать все вспомогательные окна
            window.Show(true);
        }
    }
    
    
    Private Sub ShowAllServiceWindows()
        'Перебор всех вспомогательных окон в приложении
        For Each window As ServiceWindows In Application.Window.ServiceWindows
            'Показать все вспомогательные окна
            window.Show(True)
        Next
    End Sub

В настоящее время язык не поддерживается либо пример кода отсутствует.
