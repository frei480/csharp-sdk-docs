# GraphicsSetColorLock - метод  
    
Установка режима запрета смены цвета (одноцветный режим)

**Пространство имён:** [TFlex.Drawing](N_TFlex_Drawing.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public bool SetColorLock(
	bool lock
)
```




#### Параметры

lock [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    true, если необходимо установить режим запрета смены цвета; false, если необходимо отменить этот режим

#### Возвращаемое значение

[Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)Режим запрета смены цвета, установленный до вызова данного метода
