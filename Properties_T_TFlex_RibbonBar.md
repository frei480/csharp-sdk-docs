|  | [Tab](P_TFlex_RibbonBar_Tab.md) | i-ая вкладка этой ленты |
|  | [TabCount](P_TFlex_RibbonBar_TabCount.md) | Количество вкладок в этой ленте |
|  | [Tabs](P_TFlex_RibbonBar_Tabs.md) | Вкладки этой ленты |
