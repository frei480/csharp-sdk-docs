# LocalizationMode - перечисление  
    
Режим загрузки строк локализации

**Пространство имён:** [TFlex.Dialogs](N_TFlex_Dialogs.md)**Сборка:** TFlexCommandAPI (в TFlexCommandAPI.dll) Версия: 17.1.20.0

```csharp
public enum LocalizationMode
```
```vb
Public Enumeration LocalizationMode
```
```cpp
public enum class LocalizationMode
```


| Имя члена | Значение | Описание |
| --- | --- | --- |
| Disable | 0 | Отключить локализацию |
| Normal | 1 | Режим по умолчанию |
| ByRealFullId | 2 | Локализация по полному ID элемента |
| ByCustomId | 3 | Локализация по отдельно заданному ключу |
| ByCustomIdInherit | 4 | Локализация по отдельно заданному ключу, который также будет унаследован дочерними элементами. |
