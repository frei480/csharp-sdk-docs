|  | [Keywords](P_TFlex_Model_DocumentProperties_Keywords.md) | Свойства файла->"Информация"->Ключи |

|  | [RevNum](P_TFlex_Model_DocumentProperties_RevNum.md) | Свойства файла->"Статистика"->Номер варианта |
|  | [Subject](P_TFlex_Model_DocumentProperties_Subject.md) | Свойства файла->"Информация"->Тема |
|  | [Title](P_TFlex_Model_DocumentProperties_Title.md) | Свойства файла->"Информация"->Название |
