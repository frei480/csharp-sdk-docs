# GeometryDispose(Boolean) - метод  
    
Освобождает неуправляемые ресурсы, используемые объектом [Geometry](T_TFlex_Model_Model3D_Visual_Manipulators_Geometry.md), а при необходимости освобождает также управляемые ресурсы

**Пространство имён:** [TFlex.Model.Model3D.Visual.Manipulators](N_TFlex_Model_Model3D_Visual_Manipulators.md)**Сборка:** TFlexAPI3D (в TFlexAPI3D.dll) Версия: 17.1.20.0

#### Параметры

A_0 [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    Значение true позволяет освободить управляемые и неуправляемые ресурсы; значение false позволяет освободить только неуправляемые ресурсы.
