# OpenDocumentOptions - свойства  
  
---  
  
Тип [OpenDocumentOptions](T_TFlex_OpenDocumentOptions.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Cancellable](P_TFlex_OpenDocumentOptions_Cancellable.md) | Указывает, должно ли выводиться окно отмены команды |
|  | [CancellationToken](P_TFlex_OpenDocumentOptions_CancellationToken.md) | Объект для уведомления о необходимости отмены команды |
|  | [FileLinksRefreshMode](P_TFlex_OpenDocumentOptions_FileLinksRefreshMode.md) | Режим обновления файловых ссылок |
|  | [ForceNewObject](P_TFlex_OpenDocumentOptions_ForceNewObject.md) | Создавать новый объект [Document](T_TFlex_Model_Document.md), даже если файл уже открыт |
|  | [Import](P_TFlex_OpenDocumentOptions_Import.md) | Импортировать файлы в форматах сторонних приложений |
|  | [ReadOnly](P_TFlex_OpenDocumentOptions_ReadOnly.md) | Открыть документ только для чтения |
|  | [ThrowOnError](P_TFlex_OpenDocumentOptions_ThrowOnError.md) | Выбрасывать исключение при возникновении ошибок |
|  | [Visible](P_TFlex_OpenDocumentOptions_Visible.md) | Указывает, должен ли открытый документ быть видимым |
