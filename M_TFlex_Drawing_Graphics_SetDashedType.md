# GraphicsSetDashedType - метод  
    
Установка прорисовки шрихового типа линии

**Пространство имён:** [TFlex.Drawing](N_TFlex_Drawing.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public bool SetDashedType(
	bool dashed
)
```
```vb
Public Function SetDashedType ( 
	dashed As Boolean
) As Boolean
```
```cpp
public:
bool SetDashedType(
	bool dashed
)
```


#### Параметры

dashed [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    true, если необходимо рисовать штриховую линию; false для сплошного типа линии

#### Возвращаемое значение

[Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)true, если до вызова была установлена прорисовка штриховой линии; false для сплошного типа линии
