# TermGroupRemove - метод

**Пространство имён:** [TFlex.Model.Data.Filters](N_TFlex_Model_Data_Filters.md)**Сборка:** TFlexAPIData (в TFlexAPIData.dll) Версия: 17.1.20.0 (17.1.20.0)

```csharp
public bool Remove(
	TermGroupItem item
)
```




#### Параметры

item [TermGroupItem](T_TFlex_Model_Data_Filters_TermGroupItem.md)
    Элемент для удаления

#### Возвращаемое значение

[Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)Значение true, если элемент успешно удален из группы; в противном случае - значение false
