# ApplicationSessionSetup - свойства  
  
---  
  
Тип [ApplicationSessionSetup](T_TFlex_ApplicationSessionSetup.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [DOCsAPIVersion](P_TFlex_ApplicationSessionSetup_DOCsAPIVersion.md) | **Устарело.** Номер версии T-FLEX DOCs, с которой нужно установить интеграцию |
|  | [DOCsIntegrationRule](P_TFlex_ApplicationSessionSetup_DOCsIntegrationRule.md) | Название правила настройки интеграции в T-FLEX DOCs |
|  | [DOCsLanguage](P_TFlex_ApplicationSessionSetup_DOCsLanguage.md) | Язык продукта T-FLEX DOCs, с которым необходимо установить интеграцию |
|  | [DOCsProductName](P_TFlex_ApplicationSessionSetup_DOCsProductName.md) | Название продукта T-FLEX DOCs, с которым необходимо установить интеграцию |
|  | [DOCsRegistryKeyName](P_TFlex_ApplicationSessionSetup_DOCsRegistryKeyName.md) | Ключ реестра продукта T-FLEX DOCs, с которым необходимо установить интеграцию |
|  | [Enable3D](P_TFlex_ApplicationSessionSetup_Enable3D.md) | Загрузить модуль работы с 3D информацией |
|  | [EnableDOCs](P_TFlex_ApplicationSessionSetup_EnableDOCs.md) | Разрешить интеграцию с T-FLEX DOCs |
|  | [EnableMacros](P_TFlex_ApplicationSessionSetup_EnableMacros.md) | Разрешить работу макросов |
|  | [PromptToSaveModifiedDocuments](P_TFlex_ApplicationSessionSetup_PromptToSaveModifiedDocuments.md) | Разрешить появление вопроса о необходимости сохранения изменённого документа |
|  | [ProtectionLicense](P_TFlex_ApplicationSessionSetup_ProtectionLicense.md) | Тип лицензии, который необходимо использовать при инициализации |
|  | [ReadOnly](P_TFlex_ApplicationSessionSetup_ReadOnly.md) | Режим работы "Только чтение" |
