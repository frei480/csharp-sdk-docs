# GraphicsSetLineWidth - метод  
    
Установка толщины линии для функций вывода линий, полилиний, текстов и других линейных объектов

**Пространство имён:** [TFlex.Drawing](N_TFlex_Drawing.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public double SetLineWidth(
	double width
)
```




#### Параметры

width [Double](https://learn.microsoft.com/dotnet/api/system.double)
    Значение толщины линий

#### Возвращаемое значение

[Double](https://learn.microsoft.com/dotnet/api/system.double)Толщина линий, которая была установлена до вызова данного метода
