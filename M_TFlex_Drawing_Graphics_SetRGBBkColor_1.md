```csharp
public int SetRGBBkColor(

	int b
)
```




#### Параметры

r [Int32](https://learn.microsoft.com/dotnet/api/system.int32)
    Красная составляющия (от 0 до 255)
g [Int32](https://learn.microsoft.com/dotnet/api/system.int32)
    Зелёная составляющия (от 0 до 255)
b [Int32](https://learn.microsoft.com/dotnet/api/system.int32)
    Синяя составляющия (от 0 до 255)

#### Возвращаемое значение

[Int32](https://learn.microsoft.com/dotnet/api/system.int32)Цвет фона, который был установлен до вызова данного метода
