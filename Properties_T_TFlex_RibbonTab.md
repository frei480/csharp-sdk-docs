# RibbonTab - свойства

|  | [GroupCount](P_TFlex_RibbonTab_GroupCount.md) | Количество групп в этой вкладке |
|  | [Groups](P_TFlex_RibbonTab_Groups.md) | Группы этой вкладки |
|  | [GUID](P_TFlex_RibbonTab_GUID.md) | Идентификатор вкладки |
|  | [ID](P_TFlex_RibbonTab_ID.md) | Идентификатор вкладки |
