# База знаний по классам в пространстве имён TFlex.Model.Model3D

## Point3D

**Описание:** Класс представляет точку в трёхмерном пространстве с координатами X, Y, Z.

**Основные свойства:**
- `X`: double - координата X
- `Y`: double - координата Y  
- `Z`: double - координата Z

**Основные методы:**
- Конструкторы: `Point3D()`, `Point3D(double x, double y, double z)`
- Операторы: `+`, `-`, `*` (умножение на матрицу), `==`, `!=`

**Использование:** Для задания позиций, origins, точек привязки в 3D-модели.

## Direction

**Описание:** Класс представляет направление вектора в 3D-пространстве.

**Основные свойства:**
- `X`: double - компонента X
- `Y`: double - компонента Y
- `Z`: double - компонента Z

**Основные методы:**
- `Normalize()`: нормализация вектора
- `Length`: double - длина вектора
- Конструкторы: `Direction()`, `Direction(double x, double y, double z)`

**Использование:** Для задания направлений осей, нормалей, векторов трансформаций.

## AffineTransformation

**Описание:** Класс для выполнения аффинных преобразований (перенос, вращение, масштабирование, отражение).

**Основные методы:**
- `Rotate(ModelAxis axis, double angle)`: вращение вокруг оси
- `Translate(double x, double y, double z)`: перенос
- `Scale(double x, double y, double z)`: масштабирование
- Оператор `*`: применение преобразования к точке или направлению

**Использование:** Для трансформации геометрии объектов в 3D-пространстве.

## LCS (Local Coordinate System)

**Описание:** Локальная система координат в 3D-модели.

**Основные свойства:**
- `Geometry`: доступ к геометрии LCS
  - `Origin`: Point3D - начало координат
  - `DirectionX`: Direction - направление оси X
  - `DirectionY`: Direction - направление оси Y
  - `DirectionZ`: Direction - направление оси Z

**Использование:** Для определения локальных координатных систем в сборках и деталях.

## Node3D

**Описание:** Узел в 3D-модели, представляющий точку или позицию.

**Основные свойства:**
- `Position`: Point3D - позиция узла
- `Id`: идентификатор узла

**Использование:** Для работы с ключевыми точками модели, создания размеров.

## LinearDimension3D

**Описание:** Линейный размер в 3D-пространстве.

**Основные методы:**
- `SetOrientationPlane(LCSWorkplane plane)`: установка плоскости ориентации
- `SetElements(Node3D node1, Dimension3DElementType type1, Node3D node2, Dimension3DElementType type2, DimensionUVCoords coords)`: установка элементов размера
- `SetOffsets(double offset1, double offset2, double offset3)`: установка отступов

**Использование:** Для создания размеров между элементами 3D-модели.

## TransformationGroup

**Описание:** Группа трансформаций для применения к объектам.

**Основные методы:**
- `AddMoveTransf(TransformationCoordinate coord, double value)`: добавление переноса
- `AddRotateTransf(TransformationCoordinate coord, double angle)`: добавление вращения
- `AddScaleTransf(TransformationCoordinate coord, double scale)`: добавление масштабирования
- `SetLCS(LCS lcs, bool relative)`: установка LCS

**Использование:** Для последовательного применения нескольких трансформаций.

## Parameter

**Описание:** Параметр модели с возможностью выражения.

**Основные свойства:**
- `Value`: double - численное значение
- `Expression`: string - выражение для вычисления

**Использование:** Для динамических параметров модели, масштабов, размеров.

## Operation

**Описание:** Операция в 3D-модели (выдавливание, вращение и т.д.).

**Использование:** Для добавления в проекции, работы с телами модели.

## PointsLCS

**Описание:** LCS, определяемая через точки и направления.

**Основные свойства:**
- `PointToOrigin`: Point3D - точка начала координат
- `DirectionToAxisX`: Direction - направление оси X
- `DirectionToAxisY`: Direction - направление оси Y
- `Name`: string - имя LCS

**Основные методы:**
- `SwitchAroundZ()`: переключение вокруг Z
- `SwitchAroundX()`: переключение вокруг X

**Использование:** Для создания пользовательских систем координат.

## LCSWorkplane

**Описание:** Рабочая плоскость, связанная с LCS.

**Основные свойства:**
- `LCS`: LCS - связанная система координат
- `Transformations`: TransformationGroup - трансформации

**Использование:** Для определения плоскостей видов в проекциях.

## DimensionUVCoords

**Описание:** Координаты для размещения размеров в UV-пространстве.

**Использование:** Для точного позиционирования размеров в 3D.

## ModelAxis

**Описание:** Ось в модели, определяемая точкой и направлением.

**Конструктор:** `ModelAxis(Point3D origin, Direction direction)`

**Использование:** Для определения осей вращения, симметрии.

## TransformationCoordinate

**Перечисление:** Определяет координаты для трансформаций.
- `X`, `Y`, `Z` - оси координат

## Dimension3DElementType

**Перечисление:** Типы элементов для 3D-размеров.
- `Point` - точка
- `Line` - линия
- `Circle` - окружность
- `Plane` - плоскость

## Дополнительная информация

- Все координаты и размеры используют единицы измерения документа
- Преобразования применяются в порядке добавления
- LCS могут быть вложенными в иерархии сборок
- Параметры могут содержать выражения с ссылками на другие параметры
- Размеры автоматически обновляются при изменении модели</content>
