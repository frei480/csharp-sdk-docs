|  | [Caption](P_TFlex_RibbonGroup_Caption.md) | Текст, отображаемый в группе |
|  | [ID](P_TFlex_RibbonGroup_ID.md) | Идентификатор этой группы |
|  | [Visible](P_TFlex_RibbonGroup_Visible.md) | Видимость группы |
