# ValueListMode - перечисление  
  
---  
  
Определяет доступность списка значений

**Пространство имён:** [TFlex.Dialogs](N_TFlex_Dialogs.md)**Сборка:** TFlexCommandAPI (в TFlexCommandAPI.dll) Версия: 17.1.20.0

```csharp
public enum ValueListMode
```
```vb
Public Enumeration ValueListMode
```
```cpp
public enum class ValueListMode
```


| Имя члена | Значение | Описание |
| --- | --- | --- |
| AutoVisible | 0 | Кнопка списка значений отображается если сохранено хотя бы одно значение |
| AlwaysVisible | 1 | Кнопка списка значений отображается всегда |
| Disabled | 2 | Список значений отключён |
| AutoVisibleAppend | 3 | Кнопка списка значений отображается если сохранено хотя бы одно значение и значение добавляется в конец строки |
