# PickPointResultIsOK - свойство  
    
Признак успешного завершения выбора точки или объекта. Если данный параметр равен false, то макрос или команда, вызвавшая метод PickPoint должен завершить работу. Иначе возможны сбои в работе системы.

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public bool IsOK { get; set; }
```
