# Управление видами  
  
---  
  
Класс вида документа предназначен для управления способом отображения документа в данном виде: масштабом отображения, положением изображения, наличием элементов построения и другими параметрами. Базовым классом видов документа различного типа является [View](T_TFlex_Model_View.md). Для получения текущего (активного) вида документа предназначен метод [ActiveView](P_TFlex_Model_Document_ActiveView.md). Для получения списка видов документа можно воспользоваться специальным перечислителем, который можно получить при помощи метода [Views](P_TFlex_Model_Document_Views.md).

Ниже приведен пример кода, в котором во всех видах документа гасятся линии построения.
    
    
    private void HideConstructions(Document document)
    {
        //Перебор всех видов документа в документе
        foreach (View view in document.Views)
        {
            //Погасить линии построения в данном виде документа
            view.HideConstructions = true;
        }
    }
    
    
    Private Sub HideConstructions(ByVal document As Document)
        'Перебор всех видов документа в документе
        For Each view As View In document.Views
            'Погасить линии построения в данном виде документа
            view.HideConstructions = True
        Next
    End Sub
