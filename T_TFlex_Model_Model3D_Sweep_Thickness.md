**Пространство имён:** [TFlex.Model.Model3D](N_TFlex_Model_Model3D.md)**Сборка:** TFlexAPI3D (в TFlexAPI3D.dll) Версия: 17.1.20.0

| Имя члена | Значение | Описание |
| --- | --- | --- |
| Side | 0 | Тонкостенный элемент не задаётся. Используется по умолчанию для всех типов контуров |
| Wall | 1 | Стенки нулевой толщины. Имеет смысл только для листовых контуров |
| Outward | 2 | Стенки снаружи контура. Имеет смысл только для плоских контуров |
| Inward | 3 | Стенки изнутри контура. Имеет смысл только для плоских контуров |
| Symmetrical | 4 | Стенки симметрично наружу и внутрь. Имеет смысл только для плоских контуров |
| DoubleSided | 5 | Стенки задаются разными толщинами снаружи и изнутри. Имеет смысл только для плоских контуров |
