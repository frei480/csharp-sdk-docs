# ControlRMouseUp - делегат  
    
Делегат представляющий метод, который будет обрабатывать событие, происходящее при отпускании правой кнопки мыши, когда указатель мыши находится на визуальном элементе управления.

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public delegate void ControlRMouseUp(
	Object sender,
	ControlEventArgs e
)
```
```vb
Public Delegate Sub ControlRMouseUp ( 
	sender As Object,
	e As ControlEventArgs
)
```
```cpp
public delegate void ControlRMouseUp(
	Object^ sender, 
	ControlEventArgs^ e
)
```
