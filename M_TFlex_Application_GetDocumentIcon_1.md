# ApplicationGetDocumentIcon(String, Int32) - метод  
    
Возвращает иконку документа с указанным размером, если она есть

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public static Icon GetDocumentIcon(
	string fileName,
	int iconSize
)
```




#### Параметры

fileName [String](https://learn.microsoft.com/dotnet/api/system.string)
    Имя файла документа
iconSize [Int32](https://learn.microsoft.com/dotnet/api/system.int32)
    Размер иконки

#### Возвращаемое значение

[Icon](https://learn.microsoft.com/dotnet/api/system.drawing.icon)
