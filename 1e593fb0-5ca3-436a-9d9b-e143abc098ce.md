# Управление документами  
При написании приложений с использованием Open API T-FLEX CAD возникает необходимость перечислить все открытые в системе документы, чтобы совершить для них какое-либо однотипное действие. Для этой цели предназначен специальный класс [Documents](T_TFlex_Documents.md), который можно получить при помощи метода [Documents](P_TFlex_Application_Documents.md). Этот класс реализует интерфейсы [IEnumerable](https://learn.microsoft.com/dotnet/api/system.collections.ienumerable) и [IEnumerator](https://learn.microsoft.com/dotnet/api/system.collections.ienumerator), позволяющие выполнять цикл по объектам IEnumerable-совместимого контейнера. Так как метод GetEnumerator определен как открытый, то возможно взаимодействие с типом [IEnumerator](https://learn.microsoft.com/dotnet/api/system.collections.ienumerator) напрямую или использование конструкции foreach.

Примечание  

Для получения активного (текущего) документа в системе предназанчен метод [ActiveDocument](P_TFlex_Application_ActiveDocument.md).  
Ниже приведен пример кода для закрытия всех открытых в системе документов.
```csharp
private void CloseAllDocuments()
{
	//Перебор всех документов в системе
	foreach(Document doc in Application.Documents)
	{
		//Закрыть документ
		doc.Close();
	}
}
```
