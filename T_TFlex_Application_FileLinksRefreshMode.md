# ApplicationFileLinksRefreshMode - перечисление  
  
---  
  
Режим обновления файловых ссылок

**Пространство имён:** [TFlex](N_TFlex.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

| Имя члена | Значение | Описание |
| --- | --- | --- |
| SystemDefault | 0 | Брать значение по умолчанию |
| ShowQuestion | 1 | Спросить об обновлении при открытии или закрытии документа |
| AutoRefresh | 2 | Обновлять автоматически |
| DoNotRefresh | 3 | Не обновлять |
