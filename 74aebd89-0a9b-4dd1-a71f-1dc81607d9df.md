# Экспорт документа  
    
Для экспорта документа в различных форматах предназначены ряд классов, объекты которых можно получить из документа путём вызова специальных методов. Все эти классы порождены от базового класса [ExportTo](T_TFlex_Model_ExportTo.md).

Список доступных таким образом форматов экспорта приведён в нижеследующей таблице.

| Формат экспорта | Метод получения объекта | Класс, реализующий экспорт |
| --- | --- | --- |
| AutoCAD DWG | [ExportToDWG](P_TFlex_Model_Document_ExportToDWG.md) | [ExportToDWG](T_TFlex_Model_ExportToDWG.md) |
| AutoCAD DXF | [ExportToDXF](P_TFlex_Model_Document_ExportToDXF.md) | [ExportToDXF](T_TFlex_Model_ExportToDXF.md) |
| Растровые форматы: BMP, PNG | [ExportToBitmap](P_TFlex_Model_Document_ExportToBitmap.md) | [ExportToBitmap](T_TFlex_Model_ExportToBitmap.md) |
| STEP | [ExportToSTEP](P_TFlex_Model_Document_ExportToSTEP.md) | [ExportToSTEP](T_TFlex_Model_ExportToSTEP.md) |
| IGES | [ExportToIGES](P_TFlex_Model_Document_ExportToIGES.md) | [ExportToIGES](T_TFlex_Model_ExportToIGES.md) |

Ниже приведен пример кода, в котором текущая страница экспортируется в формат PNG.
    
    
    private void BitmapExport(Document document)
    {
        //Создаем объект, экспортирующий документ в растровый формат
        ExportToBitmap export = document.ExportToBitmap;
        //Устанавливаем ширину изображения в пикселях
        export.Width = 400;
        //Устанавливаем высоту изображения в пикселях
        export.Height = 400;
        //Вызываем функцию экспорта
        export.Export("C:/bitmap.png");
    }
    
    
    Private Sub BitmapExport(ByVal document As Document)
        'Создаем объект, экспортирующий документ в растровый формат
        Dim export As ExportToBitmap
        export = document.ExportToBitmap
        'Устанавливаем ширину изображения в пикселях
        export.Width = 400
        'Устанавливаем высоту изображения в пикселях
        export.Height = 400
        'Вызываем функцию экспорта
        export.Export("C:/bitmap.png")
    End Sub
