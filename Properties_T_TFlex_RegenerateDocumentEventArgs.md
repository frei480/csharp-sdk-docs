# RegenerateDocumentEventArgs - свойства  
  
---  
  
Тип [RegenerateDocumentEventArgs](T_TFlex_RegenerateDocumentEventArgs.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Cancel](P_TFlex_PluginEventArgs_Cancel.md) | true, если действие было отменено пользователем или системой(Унаследован от [PluginEventArgs](T_TFlex_PluginEventArgs.md)) |
|  | [Document](P_TFlex_DocumentEventArgs_Document.md) | Документ, в котором произошло данное событие(Унаследован от [DocumentEventArgs](T_TFlex_DocumentEventArgs.md)) |
|  | [Is3DRegeneration](P_TFlex_RegenerateDocumentEventArgs_Is3DRegeneration.md) | Класс, содержащий данные о событии, возникшем при пересчете модели |
|  | [IsFullRegeneration](P_TFlex_RegenerateDocumentEventArgs_IsFullRegeneration.md) | true если была произведена полная регенерация модели |
|  | [UpdateFileLinksMode](P_TFlex_RegenerateDocumentEventArgs_UpdateFileLinksMode.md) | Обновить файловые ссылки |
