# AreaDispose(Boolean) - метод  
  
---  
  
Освобождает неуправляемые ресурсы, используемые объектом [Area](T_TFlex_Model_Model2D_Area.md), а при необходимости освобождает также управляемые ресурсы

**Пространство имён:** [TFlex.Model.Model2D](N_TFlex_Model_Model2D.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

#### Параметры

A_0 [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    Значение true позволяет освободить управляемые и неуправляемые ресурсы; значение false позволяет освободить только неуправляемые ресурсы.
