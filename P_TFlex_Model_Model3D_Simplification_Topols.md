# SimplificationTopols - свойство  
    
Множество геометрических элементов для удаления

```csharp
public TopolArray Topols { get; }
```




#### Значение свойства

[TopolArray](T_TFlex_Model_Model3D_TopolArray.md)

Если множество геометрических элементов пусто, то удаление выполняется на всех элементах. Все элементы должны относится к одной операции, которая возвращается в свойстве Operation
