# VariableProperties - поля  
  
---  
  
Тип [VariableProperties](T_TFlex_Model_VariableProperties.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [colorHighlight](F_TFlex_Model_VariableProperties_colorHighlight.md) | Цвет выделения при редактировании |
|  | [comboButtonPlacement](F_TFlex_Model_VariableProperties_comboButtonPlacement.md) | Положение кнопки раскрывающегося списка значений |
|  | [fEditable](F_TFlex_Model_VariableProperties_fEditable.md) | Редактирование значения допускается |
|  | [fHighlight](F_TFlex_Model_VariableProperties_fHighlight.md) | При редактировании выделять цветом |
|  | [precision](F_TFlex_Model_VariableProperties_precision.md) | Точность представления значения, только для вещественной переменной |
