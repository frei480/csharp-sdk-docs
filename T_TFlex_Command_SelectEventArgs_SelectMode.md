# SelectEventArgsSelectMode - перечисление  
    
Перечислитель режимов выбора объектов

| Имя члена | Значение | Описание |
| --- | --- | --- |
| SelectNone | 0 | Не выбирать объекты |
| SelectType | 1 | Выбирать объекты одного типа |
| SelectObject | 2 | Выбирать все объекты |
| SelectObjectElements | 3 | Выбирать составные элементы объектов |
