# ObjectIdCompareTo - метод  
    
Сравнивает текущий экземпляр с другим объектом того же типа и возвращает целое число, которое показывает, расположен ли текущий экземпляр перед, после или на той же позиции в порядке сортировки, что и другой объект.

**Пространство имён:** [TFlex.Model](N_TFlex_Model.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public int CompareTo(
	Object obj
)
```




#### Параметры

obj [Object](https://learn.microsoft.com/dotnet/api/system.object)
    Объект для сравнения с данным экземпляром. 

#### Возвращаемое значение

[Int32](https://learn.microsoft.com/dotnet/api/system.int32)

#### Реализации
