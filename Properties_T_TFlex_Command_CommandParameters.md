# CommandParameters - свойства  
    
Тип [CommandParameters](T_TFlex_Command_CommandParameters.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Bitmap](P_TFlex_Command_CommandParameters_Bitmap.md) | Отображаемое в расширенной всплывающей подсказке изображение |
|  | [Description](P_TFlex_Command_CommandParameters_Description.md) | Отображаемое во всплывающей подсказке описание команды |
|  | [ExtendedDescription](P_TFlex_Command_CommandParameters_ExtendedDescription.md) | Отображаемое в расширенной всплывающей подсказке описание команды |
|  | [IsInternal](P_TFlex_Command_CommandParameters_IsInternal.md) | Флаг, помечающий команду как используемую приложением для собственных нужд и недоступную для размещения пользователем в ленте и панелях инструментов |
|  | [LargeIcon](P_TFlex_Command_CommandParameters_LargeIcon.md) | Большая иконка команды (32х32 пикселя) |
|  | [Prompt](P_TFlex_Command_CommandParameters_Prompt.md) | Название команды |
|  | [SmallIcon](P_TFlex_Command_CommandParameters_SmallIcon.md) | Маленькая иконка команды (16х16 пикселей) |
