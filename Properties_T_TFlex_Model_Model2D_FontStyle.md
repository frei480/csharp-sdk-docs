# FontStyle - свойства  
    
Тип [FontStyle](T_TFlex_Model_Model2D_FontStyle.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Bold](P_TFlex_Model_Model2D_FontStyle_Bold.md) | Параметр "полужирный шрифт" |
|  | [ClearBackground](P_TFlex_Model_Model2D_FontStyle_ClearBackground.md) | Параметр "очистка фона" |
|  | [DefaultTracing](P_TFlex_Model_Model2D_FontStyle_DefaultTracing.md) | Использовать начертание из статуса |
|  | [Extension](P_TFlex_Model_Model2D_FontStyle_Extension.md) | Коэффициент расширения шрифта |
|  | [Fill](P_TFlex_Model_Model2D_FontStyle_Fill.md) | Параметр "заливка шрифта" |
|  | [FontName](P_TFlex_Model_Model2D_FontStyle_FontName.md) | Имя шрифта |
|  | [FontNameVariable](P_TFlex_Model_Model2D_FontStyle_FontNameVariable.md) | Переменная, задающая имя шрифта или 0 если имя шрифта является константой |
|  | [Interval](P_TFlex_Model_Model2D_FontStyle_Interval.md) | Междустрочный интервал |
|  | [Italic](P_TFlex_Model_Model2D_FontStyle_Italic.md) | Параметр "наклонный шрифт" |
|  | [LineWidth](P_TFlex_Model_Model2D_FontStyle_LineWidth.md) | Толщина линий шрифта |
|  | [Size](P_TFlex_Model_Model2D_FontStyle_Size.md) | Размер (высота) шрифта |
|  | [Spacing](P_TFlex_Model_Model2D_FontStyle_Spacing.md) | Дополнитальный интервал между символами |
|  | [Tilt](P_TFlex_Model_Model2D_FontStyle_Tilt.md) | Угол наклона шрифта |
