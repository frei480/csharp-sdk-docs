# Вспомогательные классы  
  
---  
  
Для обмена геометрическими данными в пространстве имён [TFlex.Drawing](N_TFlex_Drawing.md) реализован ряд вспомогательных структур:

  * [Point](T_TFlex_Drawing_Point.md) предназначена для хранения и действий над координатами точки, заданной при помощи вещественных значений;
  * [Rectangle](T_TFlex_Drawing_Rectangle.md) предназначена для управления координатами прямоугольника;
  * [Size](T_TFlex_Drawing_Size.md) предназначена для хранения пары значений, задающих размер (например, размеры прямоугольника), заданный в вещественных числах.
