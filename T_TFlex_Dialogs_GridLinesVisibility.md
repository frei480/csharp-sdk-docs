# GridLinesVisibility - перечисление  
    
Режим отображения линий сетки

**Пространство имён:** [TFlex.Dialogs](N_TFlex_Dialogs.md)**Сборка:** TFlexCommandAPI (в TFlexCommandAPI.dll) Версия: 17.1.20.0

```csharp
[FlagsAttribute]
public enum GridLinesVisibility
```




| Имя члена | Значение | Описание |
| --- | --- | --- |
| None | 0 | Линии не отображаются |
| Horizontal | 1 | Отображаются только горизонтальные линии |
| Vertical | 2 | Отображаются только вертикальные линии |
| All | 3 | Отображаются горизонтальные и вертикальные линии |
