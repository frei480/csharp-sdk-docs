# Создание объектов  
    
Для создания объекта T-FLEX CAD с требуемым поведением необходимо сконструировать объект, одного из классов, в соответствии со [структурой классов](classes.md) объектов документа.

Примечание  Как и при выполнении других изменений, создание объектов документа должно находиться внутри блока изменений документа, т.е. между вызовами [BeginChanges](M_TFlex_Model_Document_BeginChanges.md) и [EndChanges](Overload_TFlex_Model_Document_EndChanges.md).  
  
Ниже приведен пример кода, в котором создаётся линия изображения, проходящая через два свободных узла.
```csharp
private void HorizontalLine(Document document)
{
	//Открываем блок изменения документа
	document.BeginChanges("Создание горизонтального отрезка");
	//Создаем первый узел
	FreeNode newNode1 = new FreeNode(document, 100, 100);
	//Создаем второй узел
	FreeNode newNode2 = new FreeNode(document, 200, 100);
	//Соединяем узлы прямой
	ConstructionOutline newOutline = new ConstructionOutline(document, newNode1, newNode2);
	//Закрываем блок изменения документа
	document.EndChanges();
}
```
