# ProxyObjectEquals(ProxyObject) - метод  
    
Метод выполняет сравнение прокси объекта [ProxyObject](T_TFlex_Model_ProxyObject.md) с исходным

**Пространство имён:** [TFlex.Model](N_TFlex_Model.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
protected virtual bool Equals(
	ProxyObject object
)
```
```vb
Protected Overridable Function Equals ( 
	object As ProxyObject
) As Boolean
```
```cpp
protected:
virtual bool Equals(
	ProxyObject^ object
)
```


#### Параметры

object [ProxyObject](T_TFlex_Model_ProxyObject.md)
    Прокси объект, с которым сравнивает текущий прокси объект

#### Возвращаемое значение

[Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)true, если объекты одинаковы, в противном случае false
