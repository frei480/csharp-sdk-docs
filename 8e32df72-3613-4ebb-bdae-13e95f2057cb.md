# Общие положения  
  
---  
  
T-FLEX CAD построен на основе архитектуры "Документ" - "Вид", принятой в качестве стандарта для приложений Windows. Такая архитектура разделяет "документ", то есть данные, с которыми работает программа, и "вид", то есть способ представления этих данных пользователю. Другими словами, способ хранения данных на диске или в памяти никаким образом не влияет на их внешнее представление пользователю. Документ системы T-FLEX CAD представлен классом [Document](T_TFlex_Model_Document.md). При реализации приложения обычно объект документа передаётся системой в приложение в качестве параметра в функцию обработки команды. Для получения объекта документа можно также воспользоваться методами, реализованными в классе [Application](T_TFlex_Application.md).
