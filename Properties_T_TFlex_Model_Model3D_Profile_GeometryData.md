# GeometryData - свойства  
  
---  
  
Тип [ProfileGeometryData](T_TFlex_Model_Model3D_Profile_GeometryData.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Curve](P_TFlex_Model_Model3D_Profile_GeometryData_Curve.md) | Получить кривую, на которой лежит профиль |
|  | [CurveAxis](P_TFlex_Model_Model3D_Profile_GeometryData_CurveAxis.md) | Если профиль лежит на окружности или эллипсе, то можно получить их ось |
|  | [CurveDirection](P_TFlex_Model_Model3D_Profile_GeometryData_CurveDirection.md) | Если профиль лежит на окружности или эллипсе, то можно получить направление их осей |
|  | [CurvePoint](P_TFlex_Model_Model3D_Profile_GeometryData_CurvePoint.md) | Если профиль лежит на окружности или эллипсе, можно получить центр окружности или эллипса |
|  | [CylinderAxis](P_TFlex_Model_Model3D_Profile_GeometryData_CylinderAxis.md) | Если профиль лежит на цилиндре, то можно получить его ось |
|  | [CylinderDirection](P_TFlex_Model_Model3D_Profile_GeometryData_CylinderDirection.md) | Если профиль лежит на цилиндре, то можно получить направление его оси |
|  | [LaminarContour](P_TFlex_Model_Model3D_Profile_GeometryData_LaminarContour.md) | Получить контур - границы профиля |
|  | [LineAxis](P_TFlex_Model_Model3D_Profile_GeometryData_LineAxis.md) | Если профиль лежит на прямой, то можно получить эту прямую |
|  | [LineDirection](P_TFlex_Model_Model3D_Profile_GeometryData_LineDirection.md) | Если профиль лежит на прямой, то можно получить направление этой прямой |
|  | [Plane](P_TFlex_Model_Model3D_Profile_GeometryData_Plane.md) | Получить плоскость |
|  | [PlaneDirection](P_TFlex_Model_Model3D_Profile_GeometryData_PlaneDirection.md) | Если профиль лежит на плоскости, то можно получить направление нормали к этой плоскости |
|  | [Sheet](P_TFlex_Model_Model3D_Profile_GeometryData_Sheet.md) | Получить листовое тело профиля |
|  | [SheetBox](P_TFlex_Model_Model3D_Profile_GeometryData_SheetBox.md) | Получить границы листового профиля |
|  | [SheetContour](P_TFlex_Model_Model3D_Profile_GeometryData_SheetContour.md) | Получить контур - листовое тело |
|  | [SphereCenter](P_TFlex_Model_Model3D_Profile_GeometryData_SphereCenter.md) | Если профиль лежит на сфере, то можно получить центр сферы |
|  | [Surface](P_TFlex_Model_Model3D_Profile_GeometryData_Surface.md) | Получить поверхность, на которой лежит профиль |
|  | [ToreAxis](P_TFlex_Model_Model3D_Profile_GeometryData_ToreAxis.md) | Если профиль лежит на торе, то можно получить его ось |
|  | [ToreCenter](P_TFlex_Model_Model3D_Profile_GeometryData_ToreCenter.md) | Если профиль лежит на торе, то можно получить центр тора |
|  | [ToreDirection](P_TFlex_Model_Model3D_Profile_GeometryData_ToreDirection.md) | Если профиль лежит на торе, то можно получить направление его оси |
|  | [WireBox](P_TFlex_Model_Model3D_Profile_GeometryData_WireBox.md) | Получить границы проволчного профиля |
|  | [WireContour](P_TFlex_Model_Model3D_Profile_GeometryData_WireContour.md) | Получить контур для проволочного профиля |
