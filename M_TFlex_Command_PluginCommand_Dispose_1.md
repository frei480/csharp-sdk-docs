# PluginCommandDispose(Boolean) - метод  
  
---  
  
Освобождает неуправляемые ресурсы, используемые объектом [PluginCommand](T_TFlex_Command_PluginCommand.md), а при необходимости освобождает также управляемые ресурсы

**Пространство имён:** [TFlex.Command](N_TFlex_Command.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
protected virtual void Dispose(
	bool A_0
)
```

```vb
	A_0 As Boolean

```

```cpp
virtual void Dispose(
	bool A_0
)
```


#### Параметры

A_0 [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    Значение true позволяет освободить управляемые и неуправляемые ресурсы; значение false позволяет освободить только неуправляемые ресурсы.
