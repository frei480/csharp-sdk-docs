# Variable - свойства  
    
Тип [Variable](T_TFlex_Model_Variable.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [AssemblyVariableName](P_TFlex_Model_Variable_AssemblyVariableName.md) | Имя переменной сборки |
|  | [Attributes](P_TFlex_Model_ModelObject_Attributes.md) | **Устарело.**(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [AutoUnit](P_TFlex_Model_Variable_AutoUnit.md) | Автоматическое определение единицы измерения переменной |
|  | [Comment](P_TFlex_Model_Variable_Comment.md) | Комментарий переменной |
|  | [DisplayName](P_TFlex_Model_ModelObject_DisplayName.md) | Отображаемое название объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Document](P_TFlex_Model_ModelObject_Document.md) | Документ, являющийся родительским для данного объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [DocumentProperty](P_TFlex_Model_Variable_DocumentProperty.md) | Свойство документа |
|  | [Editable](P_TFlex_Model_ModelObject_Editable.md) | Объект находится в состоянии редактирования(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [ErrorState](P_TFlex_Model_Variable_ErrorState.md) | Состояние ошибки |
|  | [ErrorString](P_TFlex_Model_Variable_ErrorString.md) | Строка ошибки |
|  | [Expression](P_TFlex_Model_Variable_Expression.md) | Выражение, задающее значение переменной |
|  | [External](P_TFlex_Model_Variable_External.md) | true, если переменная является помеченной (внешней) |
|  | [GroupName](P_TFlex_Model_Variable_GroupName.md) | Имя группы переменной |
|  | [GroupType](P_TFlex_Model_Variable_GroupType.md) | Идентификатор типа объекта(Переопределяет [ModelObjectGroupType](P_TFlex_Model_ModelObject_GroupType.md)) |
|  | [Hidden](P_TFlex_Model_Variable_Hidden.md) | true, если переменная является скрытой |
|  | [ID](P_TFlex_Model_ModelObject_ID.md) | **Устарело.** Идентификатор объекта. Идентификатор является уникальным числом для каждого из объектов одного документа(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [IsConstant](P_TFlex_Model_Variable_IsConstant.md) | true, если переменная является константной. |
|  | [IsDisposed](P_TFlex_Model_ModelObject_IsDisposed.md) | Объект удален из модели(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [IsFunction](P_TFlex_Model_Variable_IsFunction.md) | Переменная является функцией |
|  | [IsInModelObjectGroup](P_TFlex_Model_ModelObject_IsInModelObjectGroup.md) | Является ли объект элементом группы(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [IsReal](P_TFlex_Model_Variable_IsReal.md) | true, если переменная является вещественной |
|  | [IsText](P_TFlex_Model_Variable_IsText.md) | true, если переменная является текстовой |
|  | [IsUsed](P_TFlex_Model_Variable_IsUsed.md) | true, если переменная используется (есть ссылка на неё из другого модельного объекта). |
|  | [IsVisible](P_TFlex_Model_ModelObject_IsVisible.md) | **Устарело.** Это свойство устарело и будет удалено. Пожалуйста, используйте свойство 'Visible'.(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [ListType](P_TFlex_Model_Variable_ListType.md) | Тип списка значений переменной |
|  | [ModelObjectGroup](P_TFlex_Model_ModelObject_ModelObjectGroup.md) | Группа, которая включает данный объект(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Name](P_TFlex_Model_Variable_Name.md) | Имя переменной(Переопределяет [ModelObjectName](P_TFlex_Model_ModelObject_Name.md)) |
|  | [ObjectId](P_TFlex_Model_ModelObject_ObjectId.md) | Идентификатор объекта. Идентификатор является уникальным для каждого из объектов одного документа(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [PageScale](P_TFlex_Model_ModelObject_PageScale.md) | Масштаб страницы объекта(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [Parents](P_TFlex_Model_ModelObject_Parents.md) | Контейнер родительских объектов(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
|  | [PdmConnectionDirection](P_TFlex_Model_Variable_PdmConnectionDirection.md) | Строка описания параметра PDM системы |
|  | [PdmParameterDescription](P_TFlex_Model_Variable_PdmParameterDescription.md) | Строка описания параметра PDM системы |
|  | [PdmParameterFormat](P_TFlex_Model_Variable_PdmParameterFormat.md) | Строка форматирования параметра PDM системы |
|  | [PdmParameterType](P_TFlex_Model_Variable_PdmParameterType.md) | Тип параметра PDM системы |
|  | [RealValue](P_TFlex_Model_Variable_RealValue.md) | Значение вещественной переменной |
|  | [Service](P_TFlex_Model_Variable_Service.md) | true, если переменная является вспомогательной |
|  | [TextValue](P_TFlex_Model_Variable_TextValue.md) | Значение текстовой переменной |
|  | [Tolerance](P_TFlex_Model_Variable_Tolerance.md) | Управление допуском переменной |
|  | [Unit](P_TFlex_Model_Variable_Unit.md) | Комментарий переменной |
|  | [ValueListCount](P_TFlex_Model_Variable_ValueListCount.md) | Количество строк в списке значений |
|  | [Visible](P_TFlex_Model_ModelObject_Visible.md) | Является ли объект видимым(Унаследован от [ModelObject](T_TFlex_Model_ModelObject.md)) |
