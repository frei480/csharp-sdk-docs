# PickPointEventArgs - свойства  
  
---  
  
Тип [PickPointEventArgs](T_TFlex_PickPointEventArgs.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [View](P_TFlex_PickPointEventArgs_View.md) | Вид (окно) в котором был произведён выбор точки |
|  | [x](P_TFlex_PickPointEventArgs_x.md) | Значение координаты X в целочисленных координатах окна |
|  | [y](P_TFlex_PickPointEventArgs_y.md) | Значение координаты Y в целочисленных координатах окна |
