|  | [SubType](P_TFlex_Model_ReferenceDatabase_SubType.md) | Подтип базы данных(Переопределяет [DatabaseSubType](P_TFlex_Model_Database_SubType.md)) |

|  | [UpdateMode](P_TFlex_Model_ReferenceDatabase_UpdateMode.md) | Режим обновления |
