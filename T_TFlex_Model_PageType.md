# PageType - перечисление  
    
Перечислитель типов страниц документа

**Пространство имён:** [TFlex.Model](N_TFlex_Model.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
public enum PageType
```
```vb

```

| Имя члена | Значение | Описание |
| --- | --- | --- |
| Normal | 0 | Обычная |
| Workplane | 1 | Страница рабочей плоскости |
| Dialog | 2 | Страница диалога с элементами управления |
| Auxiliary | 3 | Вспомогательная |
| Text | 4 | Текстовый документ |
| BillOfMaterials | 5 | Спецификация |
| BillOfMatrials | 5 | **Устарело.** Спецификация |
| Circuit | 6 | Электрическая схема |
| Projection | 7 | Проекция. Вспомогательная страница для проекций невидимая пользователям. Нельзя использовать для выставления типа странице. |
