# База знаний по классам в пространстве имён TFlex.Model.Model2D

## Projection

**Описание:** Базовый класс для всех типов проекций 3D-моделей на 2D-плоскость чертежа.

**Основные методы:**
- `AddBody(Operation operation)`: добавление тела операции в проекцию
- `SetViewType(ProjectionViewType type)`: установка типа вида (Front, Top, Side, etc.)
- `SetViewWorkplane(LCSWorkplane plane)`: установка рабочей плоскости вида
- `Scale`: Parameter - масштаб проекции
- `SetTiePoint(double x, double y)`: установка точки привязки на листе

**Использование:** Для создания видов и сечений 3D-моделей на чертеже.

## SimpleDrawingProjection

**Описание:** Класс для создания простых проекций без сложных настроек сечений.

**Наследование:** От Projection

**Особенности:**
- Подходит для основных видов (фронтальный, вид сверху, сбоку)
- Простая настройка через рабочие плоскости
- Автоматическое определение границ проекции

**Использование:** Для стандартных инженерных чертежей с видами модели.

## FreeNode

**Описание:** Свободный узел в 2D-пространстве, не привязанный к геометрии модели.

**Основные свойства:**
- `X`: double - координата X
- `Y`: double - координата Y

**Основные конструкторы:**
- `FreeNode(Document doc)`: создание в начале координат
- `FreeNode(Document doc, double x, double y)`: создание с координатами

**Использование:** Для создания вспомогательных точек, центров размеров, опорных точек.

## LinearDimension

**Описание:** Линейный размер между двумя элементами в 2D-пространстве.

**Основные методы:**
- `SetTwoNodes(FreeNode node1, FreeNode node2, DimensionAlignType alignType)`: установка узлов и типа выравнивания
- `SetOffsets(double? horiz, double? vert, double? parallel, double ext1, double ext2, double ext3)`: установка отступов размера
- `ValueType`: DimensionValueType - тип значения размера
- `Value`: double - численное значение (для Manual)
- `Parameter`: Parameter - параметр (для Parameter)

**Использование:** Для простановки размеров на чертежах между любыми точками.

## DimensionAlignType

**Перечисление типов выравнивания размеров:**

- `Horizontal` - горизонтальный размер
- `Vertical` - вертикальный размер  
- `Parallel` - параллельный (произвольное направление)
- `Aligned` - выровненный по элементам

**Использование:** Определяет ориентацию размерной линии относительно измеряемых элементов.

## DimensionValueType

**Перечисление типов значений размеров:**

- `Auto` - автоматическое значение (рассчитывается системой)
- `Manual` - ручное фиксированное значение
- `Parameter` - значение из параметра модели

**Использование:** Управление способом определения значения размера.

## Дополнительная информация

### Координатная система 2D
- X растет вправо, Y растет вверх
- Координаты в единицах документа
- Начало координат обычно в левом нижнем углу листа

### Масштабирование
- Масштаб проекции влияет на все размеры
- Рекомендуется использовать степени 2 (1:1, 1:2, 1:5, 1:10)
- Автомасштабирование: `ScaleFitToPageSize(width, height)`

### Размеры
- Отступы: положительные значения - наружу, отрицательные - внутрь
- Расширения: дополнительные линии от размера к элементам
- Параметры размеров могут содержать выражения

### Типы проекций
- `SimpleDrawingProjection` - базовая проекция
- `SectionProjection` - сечение
- `DetailProjection` - выносной элемент
- `AuxiliaryProjection` - дополнительный вид

### Работа с листом
- `ActivePage` - текущая страница документа
- Границы листа: `Left`, `Right`, `Top`, `Bottom`
- Единицы измерения: `Properties.Paper.Width/Height`</content>
