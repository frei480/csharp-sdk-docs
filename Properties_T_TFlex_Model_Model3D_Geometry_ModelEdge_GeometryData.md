# GeometryData - свойства  
    
Тип [ModelEdgeGeometryData](T_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [ArcTangentIntersectPoint](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_ArcTangentIntersectPoint.md) | Если ребро лежит на дуге ( части окружности ), то можно получить точку пересечания касательных на концах дуги |
|  | [Axis](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Axis.md) | Если ребро лежит на оси, то можно получить её направление |
|  | [Contour](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Contour.md) | Получить контур |
|  | [Curve](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Curve.md) | Получить кривую, на которой лежит кривая |
|  | [CurvePoint](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_CurvePoint.md) | Если ребро лежит на окружности или эллипсе, то можно получить их центр |
|  | [Direction](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Direction.md) | В большинстве случаев ребро имеет не более одного направления. Выбрать любое возможное направление |
|  | [LineDirection](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_LineDirection.md) | Если ребро лежит на прямой, то можно получить её направление |
|  | [MiddlePoint](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_MiddlePoint.md) | Получить точку в середине ребра |
|  | [Plane](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Plane.md) | Если ребро лежит в плоскости, то можно получить эту плоскость |
|  | [PlaneDirection](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_PlaneDirection.md) | Если ребро лежит на окружности или эллипсе, то можно получить направление их осей |
|  | [Surface](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Surface.md) | Если ребро лежит в плоскости, то можно получить эту плоскость |
|  | [Wire](P_TFlex_Model_Model3D_Geometry_ModelEdge_GeometryData_Wire.md) | Получить проволочную модель |
