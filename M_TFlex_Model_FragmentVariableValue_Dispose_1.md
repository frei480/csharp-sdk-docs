# FragmentVariableValueDispose(Boolean) - метод  
    
Освобождает неуправляемые ресурсы, используемые объектом [FragmentVariableValue](T_TFlex_Model_FragmentVariableValue.md), а при необходимости освобождает также управляемые ресурсы

**Пространство имён:** [TFlex.Model](N_TFlex_Model.md)**Сборка:** TFlexAPI (в TFlexAPI.dll) Версия: 17.1.20.0

```csharp
protected virtual void Dispose(
	bool A_0
)
```




#### Параметры

A_0 [Boolean](https://learn.microsoft.com/dotnet/api/system.boolean)
    Значение true позволяет освободить управляемые и неуправляемые ресурсы; значение false позволяет освободить только неуправляемые ресурсы.
