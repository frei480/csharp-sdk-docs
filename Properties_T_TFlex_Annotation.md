# Annotation - свойства  
    
Тип [Annotation](T_TFlex_Annotation.md) предоставляет следующие члены.

|  | Имя | Описание |
| --- | --- | --- |
|  | [Comment](P_TFlex_Annotation_Comment.md) | Комментарий к аннотации |
|  | [FileName](P_TFlex_Annotation_FileName.md) | Путь к файлу аннотации относительно файла аннотируемого документа |
|  | [UserName](P_TFlex_Annotation_UserName.md) | Имя пользователя, создавшего аннотацию |
|  | [Version](P_TFlex_Annotation_Version.md) | Версия аннотации |
|  | [Visible](P_TFlex_Annotation_Visible.md) | true, если эта аннотация должна быть видна при работе с аннотируемым документом, иначе false |
